# German Cloud

Es un proyecto hecho con: 
* Express
* Tailwind
* EJS

## Como levantar el proyecto en local 

### Instalar dependencias 

```bash
npm run install
```

### Correr en modo desarrollo

```bash
npm run watch
```

### Buildear

Tailwind genera el archivo de css que usaran las vistas

```bash
npm run build:css
```
